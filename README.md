# SDL2 Pong

A [Pong] game written in **C** using [SDL2].

<img src=".screenshots/start.png" width="400px">

## Dependencies

This game is being develop on a [Debian 11] environment.

* libsdl2-dev
* libsdl2-gfx-dev
* libsdl2-image-dev
* libsdl2-mixer-dev
* libsdl2-ttf-dev

## Build

To build the game run:

```sh
$ make all
```

## Running

To run the game:

```sh
$ bin/game
$ # or
$ make run
```
